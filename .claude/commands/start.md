Ты начинаешь рабочую сессию по проекту **fastcard**. Выполни шаги по порядку и общайся по-русски.

## 1. Прочитай контекст
Прочитай эти файлы (через инструмент Read):
- `ROADMAP.md` — общий план работ и статусы задач.
- `.claude/sessions/LAST_SESSION.md` — итоги прошлой сессии (напоминание: где остановились).
- При необходимости `TZ_fastcard.md` — техническое задание (источник истины).

## 2. Покажи пользователю краткую сводку
Выведи короткое сообщение на русском:
- **Где остановились** — 1–2 строки из `LAST_SESSION.md`.
- **Прогресс** — сколько задач готово / в работе / осталось по `ROADMAP.md` (посчитай по меткам `[x]`, `[~]`, `[ ]`).
- **Открытые вопросы / блокеры** — задачи с меткой `[!]`, если есть.
- **Следующий шаг** — конкретная следующая задача из roadmap (рекомендация).

## 3. Уточни и приступай
Спроси, начинать ли с рекомендованной задачи или пользователь хочет другую.
Когда задача выбрана:
- отметь её в `ROADMAP.md` как `[~]` (в работе);
- приступай к выполнению.

$ARGUMENTS

> Не забудь: в конце сессии пользователь вызовет `/stop`, чтобы сохранить итоги. Если по ходу
> работы что-то важное решено или заблокировано — держи это в уме для итогов.
