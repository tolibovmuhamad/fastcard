description: Завершить сессию по fastcard — подвести итоги, обновить roadmap и сохранить напоминание
---

Ты завершаешь рабочую сессию по проекту **fastcard**. Подведи итоги и сохрани их для следующей сессии.
Общайся по-русски.

## 1. Определи дату и номер сессии
- Узнай текущую дату: выполни `Get-Date -Format 'yyyy-MM-dd'` (PowerShell).
- Узнай номер прошлой сессии из `.claude/sessions/LAST_SESSION.md` (поле «Сессия №») и прибавь 1.

## 2. Обнови ROADMAP.md
Просмотри, что было сделано за эту сессию, и обнови статусы в `ROADMAP.md`:
- завершённые задачи → `[x]`;
- начатые, но не доделанные → `[~]`;
- появившиеся блокеры/вопросы → `[!]` (при необходимости допиши пояснение в строке задачи).

## 3. Перезапиши .claude/sessions/LAST_SESSION.md
Полностью перезапиши файл по шаблону (это «напоминание» для команды `/start`):

```
# Итоги последней сессии

**Дата:** <дата>
**Сессия №:** <номер>

## Что сделано
- <пункты: что реально сделано за сессию, какие файлы/задачи затронуты>

## Где остановились
- <в каком месте roadmap остановились>

## Следующий шаг (рекомендация для /start)
- <конкретная следующая задача>

## Открытые вопросы / блокеры
- <задачи [!], нерешённые вопросы; если нет — напиши «нет»>
```

## 4. Добавь запись в .claude/sessions/SESSIONS.md
Допиши **в начало** (после заголовка и `---`) краткий блок:

```
## Сессия №<номер> — <дата>
- <2–5 главных итогов сессии>
```

## 5. Короткий отчёт пользователю
Выведи на русском: что сохранено, текущий прогресс по roadmap и какой следующий шаг при `/start`.
